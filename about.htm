Welcome to SQLSimpl!

SQLSimpl is an open-source website and lightweight web app designed to make learning SQL accessible and approachable for beginners.
The project breaks down complex SQL concepts into digestible modules with step-by-step instructions, visual aids, and interactive quizzes.

SQLSimpl aims to eliminate the intimidation factor when learning SQL by:

  • Breaking complex concepts into easy-to-understand modules
  • Using relatable metaphors and examples (like comparing databases to spreadsheets)
  • Including interactive elements to reinforce learning
  • Creating a friendly, approachable tone and language throughout the curriculum

This website is specifically designed for people who:

  • Feel intimidated by technical topics but want to challenge themselves
  • Need to learn SQL for job opportunities or career advancement
  • Prefer learning at their own pace in a structured environment
  • Appreciate a simple, uncluttered interface when they learn online

I sincerely hope you learn something with SQLSimpl!

I love making knowledge more accessible through instructional design and barrier-free deasign principles. Besides being a school project and portfolio piece for UX and accessible design, SQLSimpl has been so much fun to make and I really do hope you have as much fun learning SQL as I did making this website!

This website was originally created as a project for my "Designing for Interactivity" class within the Information Design program at Mount Royal University, however, I have been encouraged by my wonderful friends, instructors, and professors to continue on development of SQLSimpl post graduation.

If you would like to learn more or contribute to the development of this website and making SQL more accessible as a general educational topic, please check out my GitHub repository link in the footer of this page for more information.
