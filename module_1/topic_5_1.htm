Topic 5: Logging into MySQL

This is your window into the backend of your computer's operating system.

Your computer's operating system is structured like a tree's root system, where folders exist in other folders and "branch off" into files and other folders.

The text before the % symbol is the current folder that you are located at, where you can access files or run programs from.

This might seem intimidating, but don't worry! We just need to type in one command to log into MySQL, which I'll show you shortly.
