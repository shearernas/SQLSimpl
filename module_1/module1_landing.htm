Module 1: Introduction to MySQL

This is the first module within SQLSimpl, which is also intended to be your very first touchpoint with MySQL.

Within this module you will learn about MySQL itself, some of the history of SQL, some required theory for context (don't worry, we'll keep this part as simple as possible), and also how to install and run MySQL on macOS with step-by-step picture tutorials (Windows/Linux tutorials are in the works!)

By the end of this module, you will know:
  • What MySQL is, why its so important, and what its used for.
  • A brief history of Structured Query Language (SQL) and relational databases
  • How MySQL and relational databases work
  • How to install MySQL on macOS
  • Basic navigation within the macOS Terminal command line
