Topic 5: Logging into MySQL

After you have entered in the commands from Page 5-3, to log into MySQL, type:

mysql

The CLI will prompt you to enter your password, which is where you should type in the password that you chose for the root user during the MySQL installation process.
(Whatever you type next won't appear in the CLI on purpose, for your security so people don't steal your password)

If successful, you should see this appear within your Terminal window and the CLI prompt should change to mysql> instead of *yourUsername* @ *yourMacbook* ~ %.

This is how you know MySQL is running and waiting for you to present it with a command!
