Module 1

Congratulations on finishing the very first module of SQLSimpl!

You now know what MySQL is, why its important, how it works on your computer, as well as how to install MySQL and log into it using your command line interface.

You should now have a fully functional instance of MySQL installed and running on your computer, waiting for you to create your own database and write queries.

In Module 2, we will learn more about how to create our own database, input data into a table, use data definition language to define the properties of the table and the data within it, as well as some more fun yet simple SQL theory!
